# Weila Static Content

```text
/
├── src/
│   ├── pages/           # 页面和路由
│   │   ├── ios/         # iOS 平台相关页面
│   │   ├── android/     # Android 平台相关页面
│   │   ├── web/         # Web 平台相关页面
│   │   └── *.astro      # 通用页面
│   ├── components/      # Vue 组件
│   ├── layouts/         # Astro 布局组件
│   └── shared/          # 共享常量和工具
└── package.json
```
